# Changelog

## 0.98a-RC8-kr-preview6

- 설치/제거 BAT가 `python` 명령에만 의존하지 않도록 수정했습니다.
- BAT 실행 시 Windows Python Launcher인 `py -3`을 먼저 사용하고, 없을 때만 실제 Python 3 `python` 명령을 검증해서 사용합니다.
- `python --version`이 버전 없이 `Python`만 출력되는 Windows Store alias/PATH 문제 환경에서도 설치가 막히지 않도록 했습니다.

## 0.98a-RC8-kr-preview5

- 캠페인 대화에서 특정 문구가 수십~수백 번 반복 출력되던 문제를 수정했습니다.
- `rules.csv`, `rules_master.csv`의 잘못 분리된 CSV 필드를 정리해 대화 로딩 중 크래시가 날 수 있는 문제를 수정했습니다.
- 반복 문장을 수정한 항목들의 후속 문장을 함께 검토하고, 필요한 변수 토큰이 유지되는지 확인했습니다.
- `dspOfferOption`, `rh_confirmPayAgainSel`, `devGotCottonBookSet2`에서 선택지/디버그 문구가 비정상적으로 확장되거나 대량 개행이 들어가던 문제를 수정했습니다.
- GitHub 릴리즈에서 내려받은 압축 파일 기준으로 CSV 구조, 반복 문장, 변수 토큰 검사를 통과하도록 검증했습니다.
